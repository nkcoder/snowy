Create a PostgreSQL GUI client on Mac from scratch.

Core features:
1. Connections: 
  - Grouped by projects
  - Can be add/update/delete.
  - One project can have multiple connections: local, dev, prod etc.
  - When creating/updating a connection, can save password, can test the connection.

2. Query Editor
  - Keywords (DB keywords, DB columns/tables etc.) highlight, auto-completion with low-delay
  - Can be saved to .sql file for the connection; The saved file can be updated/deleted etc.
  - Query results are shown in grid, and result set can be pinned

3. The sidebar
  - Should be flat like DataGrip, don't like PgAdmin4 with nested layers from connection to tables
